## This is a repository for my backend practices
**by using Express, MongoDB, bcryptJS and Mongoose**

### What it does
- Basic CRUD operations
- Authentication (bcrypt)
- MongoDB connection
- API endpoints

## How to run locally
```bash
npm install
npm run dev
```
## API endpoints
```bash
POST /auth/register
POST /auth/login

GET     /todos/user/:id
POST    /todos/create
DELETE  /todos/delete/:id
PUT     /todos/done/:id
```
